4. Javascript Options

All of the options below are available to enhance the plugin usage. Change the values to customize it:

Option Type Default value Possible values Description
showSpeed Integer (ms) 300 0 or greater The speed that the navigation show up
hideSpeed Integer (ms) 300 0 or greater The speed that the navigation hide up
animation String "zoom" "none", "zoom", "slide" and "dropdown" The CSS animation effect when the navigation show up

Example of use: 


jQuery(document).ready(function(){  
    jQuery("#full-screen-nav").FSNav({
        showSpeed: 200,
        hideSpeed: 500,
        animation: "slide"
    });
});

6. Methods

Some methods are available to interact with the plugin's instances.

showNav()

Show the navigation

Example: 


$("#full-screen-nav").data("FSNav").showNav();
hideNav()

Hide the navigation

Example: 


$("#full-screen-nav").data("FSNav").hideNav();
setShowSpeed(speed)

Set the speed that the navigation show up

speed: the speed in milliseconds

Example: 


$("#full-screen-nav").data("FSNav").setShowSpeed(1000);
setHideSpeed(speed)

Set the speed that the navigation hide up

speed: the speed in milliseconds

Example: 


$("#full-screen-nav").data("FSNav").setHideSpeed(700);
setAnimation(animation)

Set the navigation animation

animation: the name of the animation ("zoom", "slide" or "dropdown")

Example: 


$("#full-screen-nav").data("FSNav").setAnimation("slide");

7. Callbacks

Are available some callback functions that could be invoked:

onShow

Custom code executed when the navigation show up

Example of use: 


jQuery(document).ready(function(){  
    jQuery("#full-screen-nav").FSNav({
        onShow: function(){
            /* your code here */
        }
    });
});
onHide

Custom code executed when the navigation hide up

Example of use: 


jQuery(document).ready(function(){  
    jQuery("#full-screen-nav").FSNav({
        onHide: function(){
            /* your code here */
        }
    });
});
